The – how we call it – Object Query Framework is what powers the Automatic Reconciliation, the Generic Query and other components. What it provides:

  • QueryHelper – a thin wrapper around the SailpointContext search() methods that simplifies programming by giving direct access to extended attributes and returning records as Map<String, Object> instead of Object[]
  • TabularData – a utility class to generate tabular representations of data (CSV and HTML reports)
  • IndexedData – a utility class for working with in-memory indexed data
  • CerberusLogic – a lightweight business rules engine usable in different ways and offering a simple syntax with powerful selectors
  • OrionQL – a data query and transformation expression engine with a simple pipeline-like syntax that allows to effortlessly express complex transformations

The framework is described in the subpages. I have added also a Regular Expressions Primer to give you a headstart into this powerful mechanism. Regular expressions are used by the framework extensively and I didn't find a suitable primer on the web to link to here.
